# Hello there! 👋

I'm a student at ZHAW (Zurich University of Applied Sciences) with a strong focus on **Spatial Data Science, GIS, and Environmental Modeling**. I am passionate about working at the intersection of natural sciences and data analysis.

Here are some of my key projects from my studies at ZHAW:

**[Modelling Wildlife Corridors](https://github.com/LukasBuchmann/PA2-Modelling_Wildlife_Corridors)**
Analysis and modeling of wildlife corridors using least-cost pathways. 
* -> **[Check out the project website (Quarto)](https://lukasbuchmann.github.io/PA2-Modelling_Wildlife_Corridors/)**

**[Point Cloud Analysis](https://github.com/LukasBuchmann/PA1---Point-Cloud)**
Processing and analysis of massive 3D point cloud datasets. This project focused on the efficient handling and computation of very large spatial datasets.
